Data dimensions

Core data dimensions

There are three core data dimensions which you will face either during data capturing or during using report generation tools. These core dimensions are Organisation unit (Where - W), period (When -W) and type of metadata like indicators, defined reports e.t.c (What - W). They are generally called 3Ws.
